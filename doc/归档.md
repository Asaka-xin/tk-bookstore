**问题**：

- 注册邮箱验证码有效期和储存

  解决：使用**单例模式**共享验证码Map列表，每次触发验证码时遍历删除过期验证码。

- 前端接收数据判断问题，

  解决：统一包装成

```json
{
    "code":0
    "msg": "状态说明"
    "data": {}
}
```

**code**: 状态码，成功返回0，错误返回相应状态码

**msg**:  状态码说明

**data**：前端需要的数据

前端只需要判断code来执行相应操作

- 后端查询失败终止时如何响应前端

  解决：判断查询失败或者前端传来的参数不对都抛出自定义异常，使用@RestControllerAdvice注解捕获自定义异常返回给前端相应错误码

- 如何接收前端传来的大量参数

  解决：新增实体dto类

- 如何把多表查询结果给前端

  解决：新增vo类
