# How to Start

**`SpringBoot Part`** 

> Check configration to file "application.properties"
>
> default port:**`1055`**
>
> Most importent that is **`DatabaseConfig`**

### Not Sure?

Go to start spring boot initialization : [SpringINIT](https://start.spring.io)

**`VUE 3 Part`**

```shell
npm run serve
```
